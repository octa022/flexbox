Uses of Flexbox

Flexbox

Flex Containers

HTML


    <div class="wrapper">
      <div class="flex-container">
        <div class="box one"> </div>
        <div class="box two"> </div>
        <div class="box three"> </div>
      </div>
    </div>

CSS

    .wrapper{
      width: 100%;
      max-width: 960px;
      margin: 0 auto;
      border: 1px dashed grey;
    }
    .flex-container {
      display: flex;
      background: #fff;
    }
    .box {
      height: 100px;
      min-width: 100px;
    }
    .one {
      background: red;
    }
    .two {
      background: blue;
    }
    .three {
      background: green;
    }
