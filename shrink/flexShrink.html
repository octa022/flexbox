Uses of Flexbox

Flexbox

Flex Shrink

HTML


      <div class="wrapper">
        <div class="flex-container">
          <div class="box one"> </div>
          <div class="box two"> </div>
          <div class="box three"> </div>
        </div>
      </div>

CSS


      .wrapper{
        width: 100%;
        max-width: 960px;
        margin: 0 auto;
        border: 1px dashed grey;
      }
      .flex-container {
        display: flex;
        background: #fff;
      }
      .box {
        height: 100px;
        width: 320px
      }
      .one {
        background: red;
        flex-shrink: 1;
      }
      .two {
        background: blue;
        flex-shrink: 2;
      }
      .three {
        background: green;
        flex-shrink: 3;
      }

wrapper min-width < 960px

min-width ~ 800px
min-width ~ 640px
